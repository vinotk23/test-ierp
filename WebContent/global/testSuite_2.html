Test Suite_1 Create ZQT1 Quotation Mahender Kumar OTO Rajesh Paturu Open
[hide] [show]
    Test Suite Steps Description
    Step 1 Bid Manager goes to Interaction Center will initiate the creation of the SAP quote (ZQT1).
    The user is required to enter: organizational data, partners, identify the end user customer,
    adds the IBM contact as member of the sales team, and payment method "7" (cash).
    In addition sets the 'Consolidated Invoice Indicator' on the quote (in the Billing section).
    Step 2 Bid Manager invokes eConfig (disconnected) to configure the Power New Box with storage peripheral,
    CSW and SWMA products.Then saves a CFReport ouput.Note: Disconnected eConfig lists the entire catalog.
    Note that the 6942TSS, 6942xxx (which are used for ISAT / CHIS) are not generated in eConfig for CND.
    Therefore, the function to send the products to ISAT / CHIS is not in place for CND.
    Step 3 For the new Box configuration: No pull of Base required.
    E-config only shows list price. E config creates CFReport and CFXML.
